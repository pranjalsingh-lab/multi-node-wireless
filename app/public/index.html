Nanoleaf · Connected Lighting

Three emulated Nordic nRF52840 boards · virtual Bluetooth LE link · live firmware sandbox
Idle

What's happening here

This lab boots three virtual Nordic nRF52840 boards and runs real firmware on emulated hardware - there are no physical fixtures anywhere. Press Start and the Smart Bulb advertises a lighting service, the Lighting Hub finds it and connects over a virtual Bluetooth LE link, and live color-temperature readings begin flowing. A third board watches a fixture's accelerometer for tilt or tampering. Each card below is one board's live serial console - read it to see exactly what that firmware is doing.

How to read & tweak it

What you're seeing

Each dark box is the raw serial output of one board. The green › lines are plain-language milestones we add on top (“connected”, “streaming…”) so you can follow along without parsing every line. The badge at the top-right of each card tracks that node's current state.

What you can change

  • Drag the Tilt sensor's X / Y / Z sliders - the orientation the firmware sees changes live, as if the fixture were tilted.
  • Hit View code on any card to read the exact firmware source and how it communicates.
  • Compile your own image and Upload .elf to run it; Default reverts.
